# markdown-it-codeability

> markdown-it plugin for codeability.

## Usage

``` js
import { CTUTOR_JSON } from './ca';

var md = require('markdown-it')()
.use(require('markdown-it-codeability'), {
    config: CTUTOR_JSON
})
```
